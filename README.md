# 🔍 Serial-Checker | A C# Tool for Serial Verification

**Features:**

**🖥️ System Components Checked:**

 - BIOS:
    - Identifying Number
    - UUID

 - CPU:
    - ProcessorID
    - Serial Number (Beta)

 - Disk Drives:
   - Serial Number
 
 - Mainboard:
   - Serial Number
  
 - RAM:
   - Serial Number
 
 - Network:
   - MAC Address

**🔖 Model Information Captured:**

- Mainboard:
  - Manufacturer
  - Product
- GPU
- CPU

**💾 Data Storage:**

- Serials are saved in a TXT file.
